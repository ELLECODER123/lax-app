📲 Installation Guide

Follow these steps to install the APK on an Android device.

⚠️

Enable Unknown Sources

Since this APK is not from the Google Play Store, you must allow installation from unknown sources before proceeding.

1

Transfer the APK to your Android device

Copy BVM-Cyber-Account-v1.0.0.apk to your phone via USB, WhatsApp, email, or Bluetooth.

2

Enable installation from Unknown Sources

Go to Settings → Security (or Apps → Special app access → Install unknown apps) and enable it for your file manager or browser.

3

Locate and tap the APK file

Open your file manager, navigate to where you saved the APK, and tap it to begin installation.

4

Tap "Install" when prompted

Android will show an install screen. Tap Install and wait a few seconds for it to complete.

5

Open the App

Find BVM Cyber Account in your app drawer and tap to launch. The app will pre-fill default business info on first run.

ℹ️

Minimum Requirements

Android 5.0 (Lollipop) or higher. Fingerprint unlock requires Android 6.0+ with enrolled fingerprint. No internet required.

🚀 First-Time Setup

Recommended configuration steps after installing the app.

1

Update Business Profile

Go to Settings → Business Information. Enter your shop name, address, phone, GST number, UPI ID, and upload your logo. This appears on every PDF invoice.

2

Set Your Currency

Go to Settings → Preferences → Currency. Default is ₹ INR. Change if required.

3

Enable App Lock (Recommended)

Go to Settings → Security → Enable App Lock, set a 4-digit PIN, and optionally enable biometric unlock.

4

Add Your Customers

Go to Customers → + Add Customer. Add name, phone, and address. Customer details auto-populate when creating invoices.

5

Create Your First Invoice

Go to Bills → + New Invoice. Select a customer, add service line items, apply discount/tax if needed, and save. Generate PDF to share with the customer.

6

Take a Backup

After entering initial data, go to Settings → Data Management → Export Backup to create and share your first backup file.

📌 Important Notes

⚠️

Data is Stored Locally on Device

All data (customers, invoices, expenses) is stored on the device only. Uninstalling the app will delete all data permanently. Always keep regular backups via Settings → Export Backup.

✅

No Internet Required

The app is fully offline. It works without any network connection. Backup files can be shared and restored via WhatsApp or email.

ℹ️

Updating the App

To update to a future version, simply install the new APK over the existing installation. Your data will be preserved as it is stored independently of the app.
